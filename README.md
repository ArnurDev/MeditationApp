## ✨ Возможности

- 🎯 **Таймер медитации** — выбор длительности 1, 3, 5, 10 минут
- 🌊 **Фоновые звуки** — дождь, океан, огонь, белый шум, мурлыканье кота
- 🌬️ **Анимация дыхания** — пульсирующий круг помогает настроиться
- 🔔 **Звук колокольчика** — мягкое оповещение о начале и конце сессии
- ⏱️ **Фоновая работа** — таймер продолжает работать при сворачивании приложения
- 📊 **Статистика** — отслеживание прогресса (в разработке)

---

## 🛠️ Стек технологий

| Технология | Назначение |
|------------|------------|
| **SwiftUI** | Интерфейс пользователя |
| **UIKit** | Кастомные компоненты (GlassBackButton) |
| **AVFoundation** | Воспроизведение звуков |
| **UserNotifications** | Локальные уведомления |
| **AudioToolbox** | Системные звуки (колокольчик) |
| **Combine** | Таймер и управление состояниями |
